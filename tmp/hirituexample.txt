「比」と「比率」は違います。
「比」はふたつのものを比べた関係を表したもので「：」をはさんで二つの数字で表します。
ご質問のケースですと、「9：14」となります。
（例えば「9：18」ならばこの関係は「1：2」と言い換えることもできます。互いに倍，半分という関係は変わらないからです。）

一方「比率」は，対等に比べるのではなく，ひとつのものを基準にして，別のひとつのものの大きさを表しています。
例えば縦に対して横はどれくらいかと言えば，14/9≒1.56です（逆に横に対して縦はどれくらいかと言えば，9/14≒0.64です）

この比率が分かれば，縦が〇㎝のとき横がどれだけかが分かります。例えば縦が9㎝ならば9×1.56で横が出るわけです。この比率が分かれば自由に拡大縮小ができるわけです。

このプログラムを作った目的について

今回ゴショウセキの篆書の比率を解析して、研究の材料にするためこの様なプログラムを作った。
比率計算のアルゴリズムは以下に記述する。

アルゴリズム

#loop start
width
hight
base = width + hight

averageWidth = width / base * 100
averageHight = hight / base * 100

averageWidthSum += averageWidth
averageHightSum += averageHight

count += 1

#loop fin

#after loop

WidthResult = averageWidthSum / count
HightResult = averageHightSum / count

この計算式はまず、縦のピクセル数と横のピクセル数を取得して、
その数値を入力として、百分率のフィルターを通して、個別の平均を出していき変数に割り当てする。
次にその個別の平均をさらに全体の変数に格納して、その後カウントしていく。

すべてのファイルの平均を取得できたら、その後全体としての平均を計算し
同じ作品の文字の平均を算定する。
この数値算定後、グラフ化するため、算定後数値をResultファイルに書き込み
